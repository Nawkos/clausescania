## 6. Troubleshooting

### Upload Issues

**Problem: "Upload failed"**

Solutions:
- Check internet connection
- Ensure file size < 10MB
- Verify file format (PDF, DOCX, TXT only)
- Remove password protection from file
- Try converting to TXT format
- Clear browser cache and retry

---

**Problem: "Cannot read document"**

Solutions:
- Ensure PDF is text-based (not scanned image)
- If scanned, use OCR software first
- Try copying text to a new TXT file
- Check if file is corrupted (open it first)

---

### Analysis Issues

**Problem: "Analysis failed"**

Solutions:
- Refresh the page and try again
- Check if file uploaded completely
- Try a smaller or simpler contract first
- Ensure stable internet connection
- Try different browser
- Contact support if persistent

---

**Problem: "Score seems incorrect"**

Solutions:
- Re-analyze for confirmation
- Review specific alerts (score is based on these)
- Remember: AI assesses legal risk, not business terms
- Consider consulting a lawyer for verification
- Report issue if clearly wrong

---

**Problem: "Analysis takes too long"**

Solutions:
- Be patient (can take up to 60 seconds)
- Check internet speed
- Large documents take longer
- Refresh and retry if >90 seconds
- Try during off-peak hours

---

### Export/Save Issues

**Problem: "Cannot export PDF"**

Solutions:
- Check browser popup blocker settings
- Allow downloads from our domain
- Try different browser
- Download instead of opening
- Clear browser cache

---

**Problem: "Saved contract disappeared"**

Solutions:
- Check if browser data was cleared
- Look in "My Contracts" section
- Saved data is local (browser-specific)
- Re-analyze if needed
- Consider exporting PDFs for backup

### Display Issues

**Problem: "Page not loading properly"**

Solutions:
- Refresh the page (F5 or Cmd+R)

**Problem: "Text too small/large"**

Solutions:

## Appendix: Glossary

**Common Legal Terms Explained:**

**Clause** - A specific section or provision in a contract

**Force Majeure** - Protection from liability during unforeseeable circumstances

**Indemnification** - Agreement to compensate for damages or losses

**Intellectual Property (IP)** - Creative works, inventions, or proprietary information

**Liability** - Legal responsibility for damages or obligations

**Non-Compete** - Restriction on working for competitors after contract ends

**Notice Period** - Required time before terminating a contract

**Remedies** - Solutions or compensation when contract is breached

**Severability** - If one clause is invalid, the rest remains enforceable

**Warranty** - Guarantee or promise about product/service quality
