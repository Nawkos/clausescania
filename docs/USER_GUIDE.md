**Q: Is ClauseScan AI a lawyer?**  
A: No. ClauseScan AI is an analysis tool, NOT a lawyer. It does NOT provide legal advice. Always consult a qualified attorney for legal matters.

## ðŸ› ï¸ Troubleshooting

### Upload Errors

**Error: "File too large"**
- Solution: Compress PDF or convert to TXT format
- Max size: 10 MB

**Error: "Unsupported file type"**
- Solution: Convert to PDF, DOCX, or TXT
- Accepted formats only

**Error: "Upload failed"**
- Solution: Check internet connection and try again
- Clear browser cache if problem persists

---

### Analysis Errors

**Error: "Analysis failed"**
- Solution: File may be corrupted or in wrong language
- Try different file format
- Ensure contract is in English

**Error: "No text detected"**
- Solution: PDF is likely a scanned image
- Convert using OCR software first
- Or retype into Word/TXT document

**Error: "Contract too long"**
- Solution: Currently limited to ~50 pages
- Break into sections and analyze separately
- Focus on key sections (liability, payment, IP, termination)

---

### Display Issues

**Problem: Results not showing**
- Clear browser cache
- Try different browser (Chrome, Firefox, Safari)
- Disable ad blockers
- Check console for errors (F12)

**Problem: Can't export PDF**
- Enable pop-ups for clausescania.lovable.app
- Try "Save As" instead of auto-download
- Use desktop browser instead of mobile
